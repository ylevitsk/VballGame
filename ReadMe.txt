========================================================================
    CONSOLE APPLICATION : Final Project Overview
========================================================================


This file contains a summary of what you will find in each of the files that
make up Final Project application.

This program is an animation of two volleyball players passing
the ball back and forth.

The user can click and drags their mouse to rotate the view.

This program also has keyboard features:
	'a' & 'd' stafe
	'w' & 's' zoom


The lighting is set to {0, 8, 4}
The camera rotation is restricted to 80 degreen up and down.
There is phong shading with one light source
Timer function is used for the animations.
